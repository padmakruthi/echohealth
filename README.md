Echo Health is a simple healthcare support system built to help families manage medicines safely.
We created this project after seeing how confusing prescriptions and medicine schedules can be, especially for elderly people at home.

What We Built

Key Features
Prescription Scan

Users can scan a prescription and see simplified details like:

medicine names

basic intake instructions

The system is designed to use OCR to read text from images (simulated in this MVP).

Medicine Scan & Verification

Users can scan a medicine strip or label to:

confirm the medicine name

see the company/manufacturer

check basic intake details

This gives users confidence that they are taking the right medicine.

Smart Scheduler & Reminders

Based on the prescription, Echo Health creates a medicine schedule and reminds users to take medicines on time.
The system is designed to support voice reminders for elderly users.

Nearby Pharmacy

If a medicine is unavailable, Echo Health helps users locate nearby pharmacies using Google Maps.

Privacy & Safety

Echo Health does not prescribe medicines.
It only explains and verifies what is already prescribed by a doctor.
User data is handled with safety and privacy in mind.
